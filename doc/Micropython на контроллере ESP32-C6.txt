## 💡 Ключевые особенности MicroPython для ESP32‑C6

| Особенность | Пояснение |
|-------------|-----------|
| **USB Serial/JTAG** | Встроенный USB‑порт выступает как последовательный порт (CDC) и интерфейс отладки. В MicroPython REPL доступен через `UART(0)` или как `sys.stdin`/`sys.stdout`. |
| **Пины UART** | По умолчанию UART0 использует GPIO16 (TX) и GPIO17 (RX). В некоторых сборках эти пины могут быть зарезервированы для REPL – проверяйте документацию платы.  |
| **Датчик температуры** | В отличие от оригинального ESP32, на C6 температура возвращается в градусах Цельсия через `esp32.mcu_temperature()`.  |
| **Отсутствие ULP** | Сопроцессор сверхнизкого потребления (ULP) на ESP32‑C6 недоступен – вызов `esp32.ULP()` вызовет ошибку.  |
| **Wi‑Fi и BLE** | Поддерживаются оба интерфейса; настройка выполняется через модуль `network`.  |

## 📦 Пример: подключение к Wi‑Fi и UART

```python
import network
import machine
import time

# Подключение к Wi‑Fi
wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect("SSID", "PASSWORD")
while not wlan.isconnected():
    time.sleep(0.5)
print("Connected:", wlan.ifconfig())

# Настройка UART1 на пинах GPIO0 (RX) и GPIO1 (TX)
uart = machine.UART(1, baudrate=115200, tx=1, rx=0)
uart.write("Hello from ESP32-C6!\n")
```
